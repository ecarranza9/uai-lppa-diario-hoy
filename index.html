Red Hat OpenShift Platform Plus añade Red Hat OpenShift Data Foundation, proporcionando un stack completo y multicloud de Kubernetes.

Red Hat ha anunciado que Red Hat OpenShift Data Foundation se incorpora a Red Hat OpenShift Platform Plus, llevando los servicios de datos, incluido el almacenamiento definido por software, a su plataforma empresarial de Kubernetes. Red Hat OpenShift Platform Plus proporciona un stack completo y multicloud de Kubernetes, para profesionales de DevSecOps y del desarrollo de aplicaciones. La incorporación de Red Hat OpenShift Data Foundation ofrece a los desarrolladores una plataforma de datos consistente con almacenamiento persistente que puede abarcar nubes e infraestructuras, combinada con capacidades de gestión de datos para los equipos de operaciones de TI.

Kubernetes es el estándar de facto en la orquestación de contenedores y proporciona un potente motor para llevar a cabo la orquestación a escala, pero requiere componentes adicionales para satisfacer completamente las necesidades de la empresa para construir y ejecutar aplicaciones modernas. «Red Hat OpenShift Platform Plus reúne las capacidades necesarias que proporcionan una forma coherente de asegurar, proteger y gestionar las aplicaciones a lo largo del ciclo de vida del software en los clústeres de Kubernetes, incluso en diferentes infraestructuras de nube», escribe Red Hat en un comunicado.

Productos de Red Hat OpenShift PLUS

  1. Red Hat OpenShift Container Platform
  2. Red Hat Advanced Cluster Security for Kubernetes
  3. Red Hat Advanced Cluster Management for Kubernetes
  4. Red Hat Quay
  5. Red Hat OpenShift Data Foundation Essentials
